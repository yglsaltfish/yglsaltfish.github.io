template

Variadic Templates

template<typename T, typename... Types>
void print(const Types&... args);    //函数能接受任意参数，并且参数可以接受任意类型

  • 可以用来写递归调用，需要写一个处理最后情况的函数。

    template<typename T, typename... Types>
void print(const T& firstArg,const Types&... args)
{
    cout <<firstArg;
    print(args...);
}

void print()
{

}

    参考tuple的实现方式。

    tuple<int,float,string> t(41 , 6.13 , "hello");

nullptr and std::nullptr_t

可以用nullptr 代替 0 或者 NULL

void f(int);
void f(void *);

f(0);       // f(int);
f(NULL);    // f(int);
f(nullptr); // f(void *);

initialization

uniform initialization

int values[]{1, 2, 3};
vector<string>{"Ber", "New"};
complex<double> c{4.0, 3.0};

initializer list

int i ; //undefined value
int j{};// j is intialized by 0

int* p; //undefined

使用的是std::initialized_list<>实现。

explicit for ctors taking more than one argument

explicit //主要使用于构造函数，取消隐式转换

=default , =delete

自定义一个ctor，编译器不会给你一个default。
如果强制加上=default，就可以重新获得并使用

class zoo
{
public:
  zoo(int i1, int i2):d1(i1), d2(i2){}
  zoo(const Zoo &) = delete;
  zoo(zoo &&) = default;    //&&右值引用
  zoo &operator=(const zoo &) = default;
  zoo &operator=(const zoo &&) = delete; //&&右值引用
  virtual ~zoo(){}

private:
  int d1, d2;
}

noexcept

异常若未被处理，会上传到调用方；若最终没有处理，会调用std::terminate()，最终执行std::abort();

对于std::vector中的move constructor and destructor，必须要加上noexcept才会被调用；在vector grow(只有 vector 和 deque)过程中，会优先调用move constructor and destructor;

override

应用于虚函数，方便排查错误（想使用多态但是使用了virtual）

final

继承体系中的最后一个

auto

decltype

1、申明return type

template<typename T1, typename T2>
auto add(T1 x, T2 y) -> decltype(x + y);

2、使用在metaprogramming

typedef typename decltype(obj)::iterator iType;

3、pass the type of a lambda

auto cmp = [](const person& p1, const person& p2){
    return p1.lastname() < p2 .lastname();
};

std::set<person, decltype(cmp)> coll(cmp);

lambda

[]{
    std::cout << "hello" << std::endl;
}(); //print hello

auto l = []{
    std::cout << "hello" << std::endl;
};
l();  //print hello

右值引用

  • Lvalue: 可以出现在operator = 左侧
  • Rvalue: 只能出现在operator = 右侧

类似于在拷贝过程中使用浅拷贝，右值引用主要使用于ctor中，只更换指针。
